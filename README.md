## 运行

- Win10 + MinGW

```sh
git clone https://github.com/miawua/MRTracing.git
cd MRTracing
mkdir build
cd build
cmake -G "MinGW Makefiles" ..
mingw32-make && raytracing.exe && start out.ppm
```
