## 运行

- Win10 + MinGW

```sh
git clone https://github.com/miawua/MRTracer.git
cd MRTracer
mkdir build
cd build
cmake -G "MinGW Makefiles" ..
mingw32-make && raytracer.exe && start out.ppm
```
